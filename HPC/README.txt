Week nine of the CMEE course
AUTHOR: Rachel Bates
DATE CREATED: 26/11/18

This directory is focussed on creating functions to run a neutral theory model on Imperial's high performance computer. It contains one R script used to answer all questions on the HPC long practical, as well as results obtained from running the script on the cluster and written answers to the long practical questions.
	
R Version used: 3.2.3

Sub-Directories:

Code:	 	All code
Data:		All files to be used as inputs for scripts
Output:		Where any files generated by scripts will be placed
Sandbox: 	Misc files and outputs used while learning and experimenting
Submission:	Where the final files will be placed for submission

Scripts:

RBates.R - Script containing all functions detailed in the long practical assessment.
